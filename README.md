# Portfolio de Cris

Portfolio personal estático hecho con HTML, CSS y JavaScript.

## Contenido

- `index.html`: estructura principal del sitio.
- `style.css`: estilos y diseño visual.
- `script.js`: interacciones de la página.
- `assets/` e `image/`: recursos visuales.

## Vista local

Abre `index.html` en el navegador para ver el proyecto.

## Publicación

Este proyecto está preparado para subirse a GitHub y publicarse con GitHub Pages.
